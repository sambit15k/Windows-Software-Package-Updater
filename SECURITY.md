# Security Policy

## Supported Versions

The following versions of the Windows Software Package Updater are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |

## Reporting a Vulnerability

We take the security of this project seriously. If you find a vulnerability, please let us know immediately.

### How to Report

Please do **not** report security vulnerabilities through public GitHub issues.

Instead, please send an email to [sambit15k@gmail.com](mailto:sambit15k@gmail.com) with:
- A description of the vulnerability.
- Steps to reproduce the issue.
- Any relevant logs or screenshots.

### Response Timeline

- We will acknowledge your report within 48 hours.
- We will provide a timeline for the fix within 1 week.
- We will notify you once the fix has been released.
